Introduction to Flow Control

In Module 1, we learned:

  • Data types
  • Operators
  • Boolean expressions

Now we use those concepts to control the execution of a program.

Flow control allows a program to:

  • Make decisions
  • Repeat actions
  • Process data automatically

For example, in business applications Python can be used to:

  • Check whether a company made profit or loss
  • Categorize sales performance
  • Process multiple transactions
  • Analyze financial records

Flow control is implemented using:

  • Decision statements (if)
  • Loops (while, for)

2.1 Branching Statements (Decision Making)

Branching allows a program to choose different actions depending on conditions.

if Statement

The if statement runs code only when the condition is True.

Example:

profit = 5000
if profit > 0:
    print("The business made a profit")

Output

The business made a profit
The condition profit > 0 is True, so the statement inside the if block executes.

if – else Statement

Used when there are two possible outcomes.

Example

profit = -2000
if profit > 0:
    print("Profit")
else:
    print("Loss")

Output

Loss
Explanation
Since profit > 0 is False, the program executes the else block.
This is useful in business systems such as determining profit or loss.

if – elif – else Statement

Used when there are multiple conditions.

Example

sales = 8500
if sales >= 10000:
    print("Excellent Sales")
elif sales >= 7000:
    print("Good Sales")
elif sales >= 4000:
    print("Average Sales")
else:
    print("Low Sales")

Output

Good Sales
Explanation
The first condition that becomes True will execute.

Nested if Statements

A nested if means an if statement inside another if statement.

Example

profit = 8000
if profit > 0:
    if profit > 5000:
        print("High Profit")
    else:
        print("Moderate Profit")
else:
    print("Business Loss")

Output

High Profit
Explanation
The outer condition checks whether the business made profit.
The inner condition checks the profit level.

Compound Conditional Expressions

Multiple conditions can be combined using logical operators.

Logical operators:

  • and
  • or
  • not

Example

age = 25
has_account = True
if age >= 18 and has_account:
    print("Customer eligible for loan")

Output

Customer eligible for loan
Both conditions must be True for the statement to execute.

2.2 Iteration (Loops)

Loops allow a program to repeat a block of code multiple times.

Loops are useful for processing:

  • Sales records
  • Transactions
  • Customer lists
  • Financial data

Python provides two main loops:

  • while loop
  • for loop

while Loop

The while loop runs as long as a condition remains True.

Example

day = 1
while day <= 3:
    print("Processing sales data for day", day)
    day += 1

Output

Processing sales data for day 1
Processing sales data for day 2
Processing sales data for day 3
The loop continues until the condition day <= 3 becomes False.

for Loop

The for loop is used when the number of iterations is known.

Example

for i in range(3):
    print("Monthly sales report", i)

Output

Monthly sales report 0
Monthly sales report 1
Monthly sales report 2
Explanation
range(3) generates values 0, 1, 2.

Loop with Conditional Logic

Loops can also contain decision statements.

Example

sales = [5000, 9000, 6500, 12000]
for s in sales:
    if s >= 8000:
        print("High sales:", s)

Output

High sales: 9000
High sales: 12000
Explanation
The loop checks each sales value and prints only those greater than or equal to 8000.

break Statement

The break statement immediately stops a loop.

Example

for i in range(5):
    if i == 3:
        break
    print(i)

Output

0
1
2
When i becomes 3, the loop stops immediately.

continue Statement

The continue statement skips the current iteration and continues the loop.

Example

for i in range(5):
    if i == 2:
        continue
    print(i)

Output

0
1
3
4
When i = 2, the loop skips printing that value.

Practice Space

Question: Which statement is used to **skip** the rest of the current iteration and move to the next one?

pass Statement

The pass statement does nothing but acts as a placeholder.

Example

for i in range(3):
    pass
print("Sales processing completed")

Output

Sales processing completed
Explanation
pass is used when a block is required but no action is needed yet.

Nested Loops

A nested loop is a loop inside another loop.

Example

for month in range(2):
    for week in range(3):
        print("Month:", month, "Week:", week)

Output

Month: 0 Week: 0
Month: 0 Week: 1
Month: 0 Week: 2
Month: 1 Week: 0
Month: 1 Week: 1
Month: 1 Week: 2
Explanation
The inner loop runs completely for each iteration of the outer loop.

Integrated Business Example

Example: Sales performance analysis.

sales = [6500, 8800, 7200, 9000, 5400]
for s in sales:
    if s >= 8000:
        print(s, "High Sales")
    else:
        print(s, "Average Sales")

Output

6500 Average Sales
8800 High Sales
7200 Average Sales
9000 High Sales
5400 Average Sales
Explanation
This example combines:
• Lists
• Loops
• Conditional statements
to analyze sales performance.
