Introduction to Input and Output

In previous modules, programs used fixed values inside the code.

However, real-world programs must:

  • Accept information from users
  • Read data from files
  • Store results permanently
  • Display information clearly

This interaction between a program and external sources is called Input and Output (I/O).

Examples in business applications:

  • Entering customer information
  • Recording sales transactions
  • Reading financial records from files
  • Generating reports

Python supports two main types of input/output:

1. Console Input and Output

2. File Input and Output

3.1 Console Input and Output

Console input allows a program to receive information directly from the user.

Console output allows the program to display results to the user.

Reading Input from Console

The input() function reads user input from the keyboard.

Example

name = input("Enter your name: ")
print("Hello", name)

Sample Run

Enter your name: Rahul
Hello Rahul
Explanation
The program waits for the user to type a value.
The value entered by the user is stored in the variable name.

input() always returns string data.

Converting Input Values

Since input() returns a string, we must convert it if we want to perform calculations.

Example

age = int(input("Enter your age: "))
print(age + 5)

Sample Run

Enter your age: 20
25
int() converts it into an integer, allowing arithmetic operations.

Multiple Inputs

Programs can accept several inputs from users.

Example

price = float(input("Enter product price: "))
quantity = int(input("Enter quantity sold: "))
total_sales = price * quantity
print("Total Sales:", total_sales)

Sample Run

Enter product price: 50
Enter quantity sold: 10
Total Sales: 500
Business example:
This can represent sales transaction calculations.

Formatted Output using format()

The format() function allows structured output.

Example

name = "Rahul"
sales = 5000
print("Employee: {}, Sales: {}".format(name, sales))

Output

Employee: Rahul, Sales: 5000
Explanation
{} are placeholders that are replaced by values passed to format().

f-Strings (Modern Method)

A simpler way to format output is using f-strings.

Example

name = "Rahul"
sales = 5000
print(f"Employee: {name}, Sales: {sales}")

Output

Employee: Rahul, Sales: 5000
This method is commonly used in modern Python programs.

Command-Line Arguments

Sometimes programs receive input when the script is executed.

This is done using the sys module.

Example

import sys
print(sys.argv)

If executed as:

python script.py Rahul 25

Output

['script.py', 'Rahul', '25']
Explanation
sys.argv stores command-line inputs in a list.
First element → script name
Remaining elements → user inputs

3.2 File Input and Output

File handling allows programs to store and retrieve data permanently.

Examples in business systems:

  • Storing transaction records
  • Saving reports
  • Reading financial datasets

Opening a File

Files must be opened before reading or writing.

Example

file = open("data.txt", "w")
file.close()
Explanation
"w" means write mode.
If the file does not exist, Python creates it.

Writing to a File

Example

file = open("data.txt", "w")
file.write("Monthly Sales Report")
file.close()

File Content

Monthly Sales Report
write() stores text inside the file.

Practice Space

Question: Which mode character is used to write to a file (overwriting existing content)?

Reading from a File

Example

file = open("data.txt", "r")
content = file.read()
print(content)
file.close()

Output

Monthly Sales Report
Explanation
"r" means read mode.
read() retrieves file content.

Appending to a File

Append mode allows adding data without deleting existing data.

Example

file = open("data.txt", "a")
file.write("\nTotal Revenue: 50000")
file.close()

Updated File

Monthly Sales Report
Total Revenue: 50000
"a" mode adds new content to the end of the file.

Using with Statement (Best Practice)

A better method is using with.

Example

with open("data.txt", "r") as file:
    content = file.read()
    print(content)
Explanation
The with statement automatically closes the file, preventing errors.

Checking if a File Exists

Before reading a file, it is safer to check if it exists.

Example

import os
print(os.path.exists("data.txt"))

Output

True
Explanation
This helps prevent FileNotFoundError.

Deleting a File

Example

import os
os.remove("data.txt")
Explanation
os.remove() permanently deletes the file.
Programs should always check if the file exists before deleting.

Integrated Business Example

Example: Storing sales data.

with open("sales.txt", "w") as file:
    file.write("5000\n7000\n9000")

with open("sales.txt", "r") as file:
    data = file.read()

print("Sales Data:")
print(data)

Output

Sales Data:
5000
7000
9000
Explanation
The program:
1. Writes sales data to a file
2. Reads the stored data
3. Displays it to the user

This simulates basic data storage for business transactions.
